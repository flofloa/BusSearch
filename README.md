# BusSearch

프로젝트명(가명) 


*프로젝트 개요
극 편리성, 맞춤형에 기반을 둔 버스 정류장 찾기 플랫폼.
지도앱을 이용해 근처 버스 정류장과 노선을 알려면 현재위치 입력 -> 도착지 입력 -> 버스 노선 선택 -> 지도 확대해서 어떻게 가나 확인 등등. 아 너무 번거롭고 자칫하여 반대 노선을 탈 수도 있다. 길안내는 디바이스가 하고 나는 부지런히 다리만 움직일 순 없는 것일까? 직관적이고 빠르고 편하게 정확하게 대중교통을 이용할 수는 없는 것인가? 기존 플랫폼의 불편함에 착안해 사용자 경험의 향상을 꾀할 플랫폼을 기획하게 되었다.


어플리케이션의 구성 및 기능
실시간 위치검색, 네비게이션, 알람, 즐겨찾기
